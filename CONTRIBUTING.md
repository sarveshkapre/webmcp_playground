# Contributing

## Development flow

1. Create a branch.
2. Make focused changes.
3. Run:

```bash
npm run typecheck
npm run build
```

4. Update docs if behavior changed.
5. Open a PR with a concise summary and test notes.

## Commit message style

Use short imperative messages, for example:

- `Add weather demo tool`
- `Document WebMCP handshake notes`
- `Refactor tool response envelope`
